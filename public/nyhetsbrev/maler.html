Automatiserte flows
Velkomstserie (3 e-poster)
Trigger: ny abonnent. E-post 1: velkommen + merkevare. E-post 2: verdipåminnelse. E-post 3: konvertering (CTA).
E-post 1 — Velkommen Header → Hero → Tekst (intro) → CTA → Footer
E-post 2 — Verdi Header → Bildebanner → Tekst (fordeler) → Tekst (sosial proof) → CTA → Footer
E-post 3 — Konvertering Header → Hero (tilbud) → Produktkort × 2 → CTA → Footer
Abandoned cart (2 e-poster)
Trigger: forlatt handlekurv. E-post 1: påminnelse (1t). E-post 2: incentiv (24t).
E-post 1 — Påminnelse Header → Tekst (du glemte noe) → Produktkort (kurv) → CTA → Footer
E-post 2 — Incentiv Header → Tekst (siste sjanse) → Produktkort (kurv) → CTA (med rabatt) → Footer
Post-purchase (2 e-poster)
Trigger: etter kjøp. E-post 1: takk + ordreinfo (umiddelbart). E-post 2: kryssalg (7d).
E-post 1 — Takk Header → Tekst (takk for bestillingen) → Tekst (hva skjer nå) → Footer
E-post 2 — Kryssalg Header → Tekst (basert på ditt kjøp) → Produktkort × 2 → CTA → Footer
Win-back (2 e-poster)
Trigger: 60+ dager inaktiv. E-post 1: gjenaktivering (60d). E-post 2: siste sjanse (90d).
E-post 1 — Vi savner deg Header → Hero (personlig) → Tekst (hva er nytt) → CTA → Footer
E-post 2 — Siste sjanse Header → Tekst (eksklusivt tilbud) → CTA (med incentiv) → Footer
